# NICAR 2026 Schedule Grid

A schedule grid for the [NICAR 2026 conference](https://www.ire.org/training/conferences/nicar-2026/) in Indianapolis, with speaker bios and session highlighting.

**Live site:** https://wongpeiting.github.io/nicar/

## Credits

- Schedule grid concept and design by [Tyler Machado](https://www.tylermachado.com/nicar-schedule/)
- Schedule data from [IRE/NICAR](https://schedules.ire.org)
- Speaker bios sourced from IRE speaker profiles

## Features added

- Speaker names displayed on each session card
- Speaker bios, headshots, expertise and social links in session detail view
- Session highlighting with localStorage persistence
- Shareable highlight links via URL hash
- Export highlighted sessions as text
- Search and filter by type/track
