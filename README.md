# NICAR 2026 Resources & Schedule

A schedule grid and resource hub for the [NICAR 2026 conference](https://www.ire.org/training/conferences/nicar-2026/) in Indianapolis, with tipsheets, speaker bios, and session bookmarking. Built on the schedule grid that [Tyler Machado](https://www.tylermachado.com/nicar-schedule/) created.

## Tipsheets & materials

42 sessions linked to tipsheets, GitHub repos, slides, Colab notebooks and tutorials — sourced from [Sharon Machlis's NICAR resources database](https://apps.machlis.com/shiny/nicar20_resources/) and personal session notes. Sessions with materials are marked with a green left border on the grid.

- **"Has tipsheets"** filter shows only sessions with linked materials
- **"More resources"** panel lists 15 general tools and resources not tied to a specific session

## Features

- Speaker names on each session card
- Speaker bios, headshots, expertise and social links in session detail view
- Session bookmarking with localStorage persistence
- Shareable bookmark links via URL hash
- Export bookmarked sessions as text
- Search and filter by type/track
